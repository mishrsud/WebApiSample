How To Run this project
1. Use the OpenSSLHowTo to create a CSR, then a certificate, then a PFX
2. Install the PFX to a certificate store in Win 7 or above (if using ECC as the key algorithm)
3. Export the PFX from store in P12 format whilst checking the option to export private key and do not delete it after export
   Provide a password for the PFX
